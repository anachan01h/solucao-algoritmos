## Pré-requisitos
- Um computador com Linux e arquitetura x64
- NASM
- GCC
- Make

## Como executar
Primeiro, execute o comando:
```
make
```

O executável será compilado na basta `bin`. Para executar com um arquivo `exemplo.txt`, basta executar o comando:
```
./bin/main exemplo.txt
```
